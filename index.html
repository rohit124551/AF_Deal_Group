📱 Progressive Web App (PWA) Features

✨ Good news! You can now install this portal as an app and use it just like a native app on your device.
  • Installable: Add this portal to your home screen for a native app experience.
  • Offline Support: Access main pages even without internet.
  • Fast Loading: Frequently visited pages load instantly.

How to Install

  1. On desktop, click the install icon in your browser's address bar (usually a plus sign or phone icon).
  2. On mobile, tap the browser menu and select Add to Home screen.

Offline Usage

  • If you lose internet, you can still open the portal and view cached pages.
  • When back online, the app will update automatically.
⚠️ Internet connection lost. Some features may not work.
